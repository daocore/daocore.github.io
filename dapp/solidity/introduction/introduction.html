    我不是中本聪 2020-12-25 Solidity

    # Solidity产生的背景

    以太坊作为当前市值第二的公链项目，很大一部分原因在于他支持智能合约。为了能够开发出强大的智能合约，以太坊团队开发了EVM即以太坊虚拟机。EVM是以太坊中智能合约的运行环境。它不仅被沙箱封装起来，事实上它被完全隔离，也就是说运行在EVM内部的代码不能接触到网络、文件系统或者其它进程。甚至智能合约与其它智能合约只有有限的接触。而Solidity就是运行在以太坊虚拟机上的语言，是以太坊团队创造的一种面向合约开发的高级语言。

    # Solidity的特点

    • Solidity是一个面向合约开发的高级语言，借鉴了其他一些高级语言如Python和JavaScript。它针对以太坊虚拟机设计，被设计成以编译的方式生成以太坊虚拟机代码。
    • Solidity语言是一门静态类型语言，支持继承，支持各种库和用户自定义的类型。
    • Solidity因为非常适合做合约开发，目前已经被很多其他区块链项目也移植到自己的平台之上，例如 Nervos 上。

    简单一句话，Solidity 就是为智能合约而生，它足够小巧足够安全，让它非常适合运行在区块链上成为智能合约语言，同时 Solidity 还为智能合约和发代币等各种实际业务场景做了语言层面的优化，这个也是其他语言所不具备的，所以说人们没有使用现有的 Java/C++ 这些语言来开发智能合约，而是专门发明了 Solidity 。它非常适合用来开发类似于投票，众筹，拍卖，多重签名钱包等各种功能。

    当前Solidity的版本已经到了0.8.7了，所以之前版本的有些内容已经更新了，虽然可以在源文件头指定编译版本，但是还是希望学习和编写的时候采用最新的版本。

    # 环境搭建

    # 1.在线开发环境

    • Remix (opens new window): 以太坊官方推出的一个 Solidity 的在线集成开发环境。提供了文件浏览器，带高亮支持的代码编辑器，调试工具，甚至有编译功能。Remix 是每个 Solidity 开发者都在用的项目。等等重要信息，是上手智能合约编程之前的必备基础。
    • Remix 中文版 (opens new window)
    • ChainIDE (opens new window)

    # 2.本地环境

    • 下载安装vscode编辑器 (opens new window)
    • 安装 nodejs/npm: node官方网站 (opens new window)下载node，推荐LTS版本，按提示完成安装，npm会同时装上。
    • 安装solidity (opens new window)扩展
    • 安装truffle (opens new window)或hardhat (opens new window)框架进行智能合约开发，Truffle文档 (opens new window)、Hardhat文档 (opens new window)、Truffle教程 (opens new window)、Hardhat入门教程 (opens new window)

    # 学习资料

    • Solidity 最新(0.8.0)中文文档 (opens new window): 登链社区维护的Solidity中文文档，更新速度快，翻译质量高，是一个非常不错的社区。也是当前笔者学习的主要平台。
    • Solidity 官方英文版 (opens new window): 以太坊官方英文文档
    • Solidity 文档中文版 (opens new window): 极客学院翻译的Solidity文档，该文档已经两年没有更新了，但是其中有一些内容可以丰富solidity的知识面。所以可以阅读一下。
    • Solidity 教程 (opens new window): 奇客谷的solidity学习教程
    • 加密僵尸 (opens new window): 是一个第三方的在线智能合约学习环境，目标是教会大家实现 DApp 游戏。这个网站的特点是自作非常精良，各种辅助学习材料丰富，并且是对 Solidity 小白友好的。
